# Security Policy

## Supported Versions

Security reports are accepted for the latest `main` branch until formal releases
begin.

## Reporting

Please do not open public issues for security-sensitive reports. Contact the
repository maintainer privately through GitHub.

## Scope

ComputeOS is a research framework. Reports involving arbitrary model code,
untrusted `trust_remote_code`, or user-supplied plugins should include a minimal
reproduction and threat model.
